# Cat Auto Catcher Install

## Mac

1. Double-click `launch-cat-autocatcher-mac.command`.
2. Use the Firefox window that opens.
3. Open the cat page and press `Start` in the floating panel.

If macOS blocks the launcher, right-click it and choose `Open`.

## Windows

1. Double-click `launch-cat-autocatcher-windows.bat`.
2. Use the Firefox window that opens.
3. Open the cat page and press `Start` in the floating panel.

## Test Page

The included `test-page.html` can confirm that the extension detects the trigger and attempts to send `cat`.
